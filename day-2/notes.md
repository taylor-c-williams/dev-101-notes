### CSS FR Grid and Flexbox
   - FR: Fractional Unit
   - Breaks up styles into proportions based on the height and width of a section

Grid ["2D"]

    display: grid;
    grid-template-rows: 1fr 1fr .5fr;

Flex (flex box) ["1D"]

    display: flex;

* Flex does not collapse/de-collapses  margins
